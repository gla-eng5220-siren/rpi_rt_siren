# GPIO buzzer

```
  --buzzer              GPIO buzzer pin
```

e.g.

```
  --buzzer 26
```

uses the buzzer connected to GPIO pin 26

# Email Alarming

_Warning: using email alarm is a little challenging to setup and need money/resource._

## Buy a domain

Purchase a domain at retailers such as (GoDaddy)[https://www.godaddy.com/]. If you don't own a domain and use personal emails instead, there's fat chance the alarming email will end up in the spam box, defeating the whole purpose.

## Configure Brevo

Go to [Brevo](https://www.brevo.com/) and setup your account for transactional email. Make sure that you

- Creates an API key
- Registered your domain for usage with transactional email API

## Configure FlameIris

These options must be set:

```
  --brevo-api-key       Brevo email alarming API key 
  --brevo-sender-email  Brevo email alarming sender email
  --brevo-to-email      Brevo email alarming receiver email 
```

and the `--brevo-sender-email` should have your domain suffix. Other options are also able for tuning, while the default will work most of the time.
