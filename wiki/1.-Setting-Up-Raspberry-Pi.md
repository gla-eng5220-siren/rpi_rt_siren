# Install Raspbian Trixie

Refer to [Official Install Guide](https://www.raspberrypi.com/documentation/computers/getting-started.html#install)

# Using Camera

We tested using the following camera:

- Innomaker-U20CAM-720P

In theory, any UVC camera is applicable.

# Using Temperature Sensor

We used the following hardware:

- BreadPi HAT
- 10KOhm Resistor
- RS Pro NTC Temperature Sensor (124-1081)

## Enable I2C

Run:

```
sudo raspi-config
```

Navigate to:

```
Interfacing Options -> I2C
```

And enable it.
